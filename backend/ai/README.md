LLM-Based Health Outbreak Prediction System

## 🚀 Overview
This system uses:
- Llama 3 (Ollama)
- Rule-based logic
- Prompt engineering

No ML/DL training required.

## ⚙️ Setup
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## 🤖 Run LLM
```bash
ollama run llama3
```

## 📡 API Endpoint
POST /predict

## 💡 Features
- Disease prediction (LLM-based)
- Risk classification
- Case estimation
- Recommendations
